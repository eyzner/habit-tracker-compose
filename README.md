# Habit Tracker Compose

Учебное Android-приложение «Трекер привычек» на Jetpack Compose.

## Стек

- Kotlin
- Jetpack Compose
- Material Design 3
- ViewModel
- minSdk 24
- GitHub Actions: `Habit Tracker CI`

## Реализовано

- экран списка привычек;
- экран создания привычки;
- добавление привычки;
- список открывается пустым, без заранее созданных карточек;
- запрет сохранения пустого названия;
- автоматический возврат к списку после сохранения;
- отображение названия привычки;
- отображение статуса на сегодня: выполнено / не выполнено;
- прогресс по каждой привычке;
- прогресс по дням недели;
- изменение статуса только для сегодняшнего дня;
- повторный клик откатывает статус сегодняшнего дня обратно;
- удаление привычки свайпом по горизонтали;
- введённое название сохраняется при смене конфигурации.
